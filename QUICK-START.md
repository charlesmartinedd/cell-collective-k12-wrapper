# Quick Start Guide - Cell Collective K-12 Wrapper

**Get running in 5 minutes!**

## Step 1: Install Dependencies
```bash
pip install flask beautifulsoup4 requests
```

## Step 2: Navigate to Project
```bash
cd C:/Users/MarieLexisDad/projects/cell-collective-k12-wrapper
```

## Step 3: Start Proxy Server
```bash
python src/proxy_server.py
```

## Step 4: Open Browser
Navigate to: **http://localhost:5000/research/dashboard/**

## Step 5: Verify K-12 Modifications

### Check for:
- Custom purple header with "Student Edition"
- No sign-in/sign-up buttons
- Enhanced green "New Model" button
- Console message: "Cell Collective K-12 Wrapper v1.0 initialized"
